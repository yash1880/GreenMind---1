# GreenMind

A responsive landing page for a plant shop with a continuous auto-scrolling customer testimonial slider.

## Features

- Responsive header and footer
- Responsive hero, product, and testimonial sections
- Auto-scrolling testimonial carousel with smooth right-to-left loop
- Pause on hover/touch for the testimonial slider
- Clean mobile stacking for footer and content

## Files

- `index.html` — main page markup
- `style.css` — responsive styling and layout
- `script.js` — page interaction logic
- `media.css` — media and breakpoint utilities
- `image/` — image assets used in the page

## How to use

1. Open `index.html` in a browser.
2. The testimonial slider will auto-scroll from right to left.
3. On mobile, the footer and carousel adapt to smaller screens.

## Notes

- The slider uses a seamless duplicated loop technique to avoid vibration.
- Footer layout is optimized for narrow viewports with full-width stacking.

## Development

To update the page:

1. Edit `index.html` for structure and markup.
2. Edit `style.css` for styling and responsive behavior.
3. Edit `script.js` for interaction logic.
